Goal-First Autonomous Engineering

VA Auto-Pilot delegates execution to top models and enforces deterministic quality gates so outcomes are auditable, reproducible, and resilient.

Design Philosophy

A protocol, not a prompt template

Do not tell agents how

Define the highest standard, hard constraints, and measurable acceptance. Leave path planning to frontier models.

Manager agent over manual orchestration

Humans define destination and boundaries. The manager agent coordinates implementation agents through deterministic CLI actions.

Closed-loop verification

Build, review, acceptance, and state updates are mandatory. Completion is machine-verified, not presentation-driven.

Model frontier by default

For high-leverage tasks, use the strongest model available. Better planning quality compounds over long loops.

Memory is explicit

State and run memory live in files, not assumptions. The loop remains recoverable after resets and handoffs.
